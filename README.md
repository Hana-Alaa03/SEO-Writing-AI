# SEO Writing AI

An advanced, AI-driven content generation engine designed for high-performance SEO. This system balances ease-of-use with professional-grade customization tools.

## System Features

The system operates in two core modes:

### 1. Basic Plan (Core Mode)
- **Autopilot Efficiency**: Requires only a Title, Keyword, location, and Brand URLs.
- **AI-Managed Strategy**: Automatically handles keyword intent, competitor research, and outline generation.
- **Rich Media**: Generates 7 default context-aware images per article.
- **SEO Ready**: Handles meta tags, internal/external linking, and semantic optimization automatically.

### 2. Advanced Customization
- **Granular Control**: Manually override Tone, Point of View (1st/2nd/3rd person), and Language.
- **Structure Controls**: Explicitly include/exclude Conclusions, FAQs, Tables, and Lists.
- **Multimodal Brand Voice**: Upload guidelines, example articles, or provide a text description.
- **Media Control**: Select image types (Infographic, Mockup, Mixed), customize quantities, and set explicit sizes (Square, Wide, Portrait).
- **Advanced SEO**: Fine-tune keyword density and control competitor analysis depth.

## Installation & Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Start the API: `python src/app/api.py`
3. Access the Dashboard: `src/app/static/index.html`

## Testing
